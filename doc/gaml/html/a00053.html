Detailed Description

template<typename MergeOutput, typename BasisRandomizer, typename ElementaryLearner>
class gaml::bag::Learner< MergeOutput, BasisRandomizer, ElementaryLearner >

This learner produces a collection of predictors by learning each one from a randomization of the input data set.
